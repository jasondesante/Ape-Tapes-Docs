# The Featured Page

The Featured Page is currently the home page when you go to The Contract Wizard.  There are 4 modules in The Featured Page:

## The Wizard's Library

This module has a list of all the contracts that are a part of The Wizard's Library on the selected network.  Clicking on the contract will load that contract's collection.  Works on Ethereum mainnet and Sepolia testnet.

## Featured Songs

This module has a list of tracks that are staked to the store.  Every track that is staked is from a contract in The Wizard's Library.  The icons in this module are songs that will play if you click them. &#x20;

## Recent Playlists

This module has a list of the most recent shared playlists that were uploaded through the playlist creator.

## Featured Collections

This module has a list of featured collections from across Ethereum.  These contracts are just a bunch I picked and put in a list.
