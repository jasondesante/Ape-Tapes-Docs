# Hotkeys

The visualizer has lots of hotkeys. 90 last time I counted. Hotkeys are case sensitive.\

### Modes

* Toggle Help Mode - Q
* Exit Mode - =
* Toggle Fullscreen - f
* Toggle UI Visibility - g
* Add Random Avatar - z



## Camera

### Orbit Camera

* Toggle Camera Orbit - e
* Lower Orbit Rotation Speed - 1&#x20;
* Flip Orbit Rotation - 2
* Raise Orbit Rotation Speed - 3(# Raises speed 100x faster)
* Set Rotation Speed to 0.002 - !
* Set Rotation Speed to 0.2 - @

### FPS Mode

* Toggle FPS Mode - r
* Movement - WASD
* Run - Shift
* Toggle Traveler's Mode - t (only in fps mode) - When enabled, the spawn point will move when you next exit FPS mode.
* Change Sound Guns - e - Guns make sounds and delete avatars when shot. Left click and right click the ground, sky, 3d model or avatars for different sounds.  Shift also makes a sound.



## Dancers

### Animations

* Next Animation - h
* Shuffle Animation - j
* Nudge Animation Speed Down 0.02 - left arrow
* Nudge Animation Speed Up 0.02 - right arrow

### Position Avatars

* Spread Out Avatars - k
* Line Up Avatars - l



## Layers

### Resize Windows

With the main visualizer layer, shader layer, and youtube layer, you can adjust the size and position of each layer. Press Command to toggle through available layers, and then adjust with the wasd keys.

When not in FPS mode you can use:

* Move position of video window - wasd
* Resize video window width and height - WASD

### Visualizer Control

* Nudge Opacity Down 0.05 - down arrow (or x when a video isn't playing)
* Nudge Opacity Up 0.05 - up arrow (or c when a video isn't playing)
* Toggle Opacity Empty - v
* Toggle Opacity Full - b
* Next Visualizer Preset - n
* Toggle Visualizer Mode Manual/Auto - m
* Change Visualizer Layer - u

#### Visualizer Pixel Ratio

* Pixel Ratio - 5 and 6
* Set Pixel Ratio to 0.01 - %
* Set Pixel Ratio to 4 - ^

### Shader Layer

* Toggle Shader Layer - 4
* Change Shader presets - Y and U
* Change Shader render quality - H and J
* Change Shader opacity - V and B
* Change Shader layer - Z

### YouTube Mode

* Toggle YouTube Mode - 8 (adds a youtube layer in the background), "control" on mac.
* Random YouTube Link - "\*", "q" - changes to a random video from a pre selected list.
* Toggle Playback Speed - "-" - toggles video playback speed from 1 to 0.25
* Change Video Layer - "y" - toggles the video layer from behind the 3d models to in front of them, "t" also when not in fps mode
* Rewind - 9, "X" - Rewinds YouTube video 10 seconds
* Fast forward - 0, "C" - Fast forwards YouTube video 10 seconds&#x20;
* Lower Video Opacity - x - Lowers YouTube video opacity 0.1
* Raise Video Opacity - c - Raises YouTube video opacity 0.1



## 3D World Stuff

* Next Sky - , (comma key)
* Next Floor - . (period key)
* Next Sky and Floor - ' (apostrophe key)
* Next Model - ; (semicolon key)
* Reset Model - : (colon key)
* Toggle Floor Visibility - / (slash key)
* Toggle Sky Visibility - ?

#### Skybox LODs

* Next Sky (high quality) - < (less than)
* Next Sky and Floor (low quality) - " (double quotes)

### Model/Prop Control

* Nudge Model Down - i ( "I" = 25x)
* Nudge Model Up - o ( "O" = 25x)
* Toggle Model PBR Lighting - p

#### Model/Prop Scale

* Model Scale Down - \[
* Model Scale Up - ]
* Model Scale Step Down 10x - {
* Model Scale Step Up 10x - }

### Other Stuff

* COMMAND - change WASD mode
* Secret - + - loads secret dance
