# Other Features

Other features of the player include:

### Offline Mode

Download all the files you play so the next time you play them you're accessing a local file instead of downloading it from the internet.  This will let you create a library of tracks you can listen to offline.

### MP3 / Wav Support

The player is smart enough to offer you the choice of speed or quality if the token has both WAV and MP3 in the metadata. This is important because they are both desirable under different circumstances.
