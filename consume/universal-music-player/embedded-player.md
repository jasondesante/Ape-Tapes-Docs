# Embedded Player

## Embedded Player

You can embed a player into your site easily. The embed link can be to a contract or wallet, and they can include filtering and sorting in the url. &#x20;

You can also embed Serverless Playlists which turns the embedded player into a serverless embedded player!

## Generating Embed Link

Click any song and there will be a button to embed the song.&#x20;

You can also take any player link and pop it into the Embed Config page in the settings.  This will generate an embed link for you and give you more options to customize.

## Configure the Embed Window

Go to the Settings page, and there will be a button called "Config Embed". &#x20;

The Embed Config page lets you configure the embedded link.  It generates a preview of the embed link right next to the input so you can tweak how things look. You can create embedded windows of players and store pages.

* You can put the url to any player page link or store page link into the "Configuration" input field.
* For player links you can use the "Player Options" to choose some special options.&#x20;

"Mini Player" will set the size of the player to a small size right away, and turns on "Auto Play".&#x20;

"Scroll to Song" will scroll the page to the correct song upon load. "Auto Play" pre loads a song into the player.  "Visualizer" automatically loads with the visualizer open.

"Playlist Number" lets you choose which track gets loaded, scrolled to.

* Iframe Dimensions lets you choose the size and scale of the embedded iframe.
* Generated URL is the url of the full page embedded link
* Embed Code contains the HTML and React code for a fully embedded player onto your own site with the dimensions you specified.
