# Arweave Explorer

The Arweave Explorer page lets you search using tags on Arweave. &#x20;

You can search audio files, playlists, and all files.

## Audio Search

hi ![](<../../../.gitbook/assets/Screenshot 2025-07-12 at 5.07.36 AM.png>)

Search by title, artist, genre, tag, isrc, uploader, and the mini time machine.  For files uploaded with The Contract Wizard, and all of Arweave.

## Playlist Search

![](<../../../.gitbook/assets/Screenshot 2025-07-12 at 5.08.06 AM (3).png>)

Search any playlist name, genre or tag. It's easy to start curating, and even easier to search once it's been curated.

## File Search

![](<../../../.gitbook/assets/Screenshot 2025-07-12 at 5.11.06 AM (1).png>)

Search any tag on any file, uploaded with The Contract Wizard, or all of Arweave.  Search by uploader, or IPFS.  All things that are tagged with every upload if you upload with The Contract Wizard.
