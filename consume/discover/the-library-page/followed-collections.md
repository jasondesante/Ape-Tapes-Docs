# Followed Collections

The collections tab shows you all the contracts, wallets, and artists you've followed.

![](<../../../.gitbook/assets/Screenshot 2025-07-12 at 4.36.52 AM (1).png>)

This page shows:

## Favorite Contracts

Any contract address that you favorite will show up here.  It will also show what chain you favorited the contract on.

## Favorite Wallets

Any wallets you favorite will show up here.  It will also show what chain it was favorited on.

## Spinamp Artists

When viewing artists from Spinamp you can favorite any artist and it will show up in this list.

### ENS Verified

Verification works slightly different for contracts and wallets.

You will see a verified checkmark next to the wallet if the wallet has the ENS set to resolve to the wallet and the wallet's primary name is set to the same ENS.

You will see a verified checkmark next to any contract that has a contract name and matching ENS primary name registered to it.  This requires the smart contract name to match the ENS.
