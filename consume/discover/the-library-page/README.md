# The Library Page

The Library Page shows your wallet as well as all the collections and tracks you've favorited and made playlists of. &#x20;

The Library is divided into 4 categories, [Wallet](your-wallet.md), [Favorites](favorite-songs.md), [Playlists](playlists.md), and [Collections](followed-collections.md).
