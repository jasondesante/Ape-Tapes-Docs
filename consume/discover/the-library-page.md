# The Library Page

The Library Page is full of all the stuff you've collected in your wallet as well as all the collections and tracks you've favorited and made playlists of.  The Library is divided into 4 categories:

## Your Wallet

This page shows all the tokens in your wallet that have music in them, and lets you play them.  In your wallet you can then filter and sort all the tokens in any way you want.  If you want to save a snapshot of your wallet with particular filters and sorting you can create a playlist and add the tracks.  You can share the link too with the filters and sorting in place.

## Favorite Songs

This page shows a playlist of the tracks you favorited.  The favorites playlist is separate from all the other custom playlists because it's always there and connected to the favorite icon.

## Playlists

This page shows a bunch of modules related to playlists. &#x20;

### Created Playlists

This lets you create a new playlist and shows you all the playlists you already created.  You also have the option to rename, delete, change the image, and publish the playlists.

### Favorite Playlists

Any playlist that you favorite will show up here.  These are playlists that have been published using The Contract Wizard's playlist creator, or created with the same standards.

### Published Playlists

This shows the list of playlists that you have created and then publicly uploaded / shared.  Anyone can view these playlists that you published. &#x20;

### Favorite Spinamp Playlists

This list is where all the Spinamp playlists that you favorite will go.

## Collections

This page shows:

### Favorite Collections

Any contract address that you favorite will show up here.  It will also show what chain you favorited the contract on.

You will see a verified checkmark next to any contract that has a contract name and matching ENS primary name registered to it.

### Favorite Wallets

Any wallets you favorite will show up here.  It will also show what chain it was favorited on.

You will see a verified checkmark next to the wallet if the wallet has the ENS set to resolve to the wallet and the wallet's primary name is set to the same ENS.

### Spinamp Artists

When viewing artists from Spinamp you can favorite any artist and it will show up in this list.
