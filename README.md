# Welcome

## What is Listen?

Listen is the open library of music that consists of all tokens and files on Ethereum and Arweave.  Listeners curate this vast library feeding value back into the system.

Listen is a portal to decentralized music streaming.  As a listener, browse and search any collection or wallet.  Create playlists that can be shared and played without a server.  An important part in building a [Decentralized Autonomous Music Network (D.A.M.N.)](introduction/welcome/d.a.m.n..md)

## What is the Contract Wizard?

The Contract Wizard is a suite of tools for artists.  The Contract Wizard is also this character who loves music, has a talking hat, and creates smart contracts in 1 click. &#x20;

<figure><img src=".gitbook/assets/TheContractWizard_ft (1) (1).gif" alt=""><figcaption></figcaption></figure>

### Tools

* [**The Audio Uploader**](create/files/audio-uploader/) properly tags all files, whether they're audio, or playlists, setting the bar for discoverability. &#x20;
* [**The Metadata Maker**](create/the-metadata-maker/) is an essential tool to upgrade audio and metadata to be as thicc as they can be.
* [**The Manifest Maker**](create/the-metadata-maker/manifest-maker/) is a powerful tool that helps you create and upgrade large collections.&#x20;
* [**The Contract Wizard**](create/tokens/the-contract-wizard/) helps Artists can create their own contracts, and manage their record labels.&#x20;

All these tools are available to be used together, or separately.  You can choose to use one tool, or use the full pathway.  [The Hat Helper](create/tokens/the-contract-wizard/hat-helper.md) is there to guide you every step along the way.

The mission is to contribute meaningfully to building a [Decentralized Autonomous Music Network](introduction/welcome/d.a.m.n..md).

[The Wizard's Library](create/tokens/the-contract-wizard/the-wizards-library.md) is a collection of independently owned record label contracts created with The Contract Wizard. Anyone can create a record label that's fully theirs. &#x20;

You can listen to music, collect and create, all at the same place.

## [721J](create/smart-contracts/721j-concepts/)

The [721J](create/smart-contracts/721j-concepts/) is a modified ERC721 that includes features such as digital masters with on chain rarity.

The Contract Wizard uses magic to create 721J contracts with ease.
