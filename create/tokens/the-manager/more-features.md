# More Features

Here are some of the other important features of The Manager page.

## Mint Copies

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-19 at 3.43.07 AM.png" alt=""><figcaption></figcaption></figure>

The owner of a contract can mint copies of any rarity.  This lets the owner set up interesting promotional rarities that are outside the normal range, creating separate branches to the rarity tree.&#x20;

The owner can also mint regular rarity copies to their friends for free with this button. This feature doesn't always need to be used for adventurous reasons, it can also be used for normal things.

## Toggle Recycling

You can toggle Recycling Mode, which unlocks the ability for holders to burn 2 old tokens to mint 1 new token.  This is an optional mode that adds a mechanic that tries to balance supply and demand.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-19 at 3.42.42 AM.png" alt=""><figcaption></figcaption></figure>

## Stake Tokens

You can stake multiple tokens at once.

## View ENS

You can view the ENS that is set to the contract address, if one is set.  If the ENS is matching the name of the contract, the collection becomes verified and receives a verified checkmark icon.

Make sure you name your contract something that you can grab the ENS for! Don't get caught slipping!

## Transfer Ownership

You can transfer the ownership of the contract to any address.

## Troubleshooting

You can delete the contract files from the Contract Wizard front end database if you think there are any issues.&#x20;
