# The Manager

## What is the Manager?

The manager page is where the owner of a contract goes to do all the managerial things.

On the Manager page, artists can mint new originals, set the price for a copy, set promos and more.  The Contract Manager is the page that only the owner of a contract has access to. From there they can manage all the settings of their contracts. &#x20;

## Functionality

The Contract Manager page has lots of info to look at, and lots of buttons to change settings.  Almost every function on the smart contract is accessible on this page.

## Tabs

The Manager is currently organized into 4 main tabs:

* Main - The main tab has most of the general settings.&#x20;
* Price - Price and royalties
* Info - Functions to view different information
* Promos - Set and view promotions

There's also the top row of minting buttons that includes the "Mint Master" and "Mint Copy" buttons, which are always there.
