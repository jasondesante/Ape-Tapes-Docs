# View Data

On The Manager Page you can quickly view the most important information as well as look up anything on chain.

## Main Tab

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-18 at 10.30.39 PM.png" alt=""><figcaption></figcaption></figure>

On the **Main** tab, you can view the: &#x20;

* Total amount of tokens
* Total amount of originals
* Name and symbol of the contract
* Contract address

## Info Tab

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-18 at 10.31.21 PM.png" alt=""><figcaption></figcaption></figure>

On the Info tab you can view a lot more data.

### Song Info

For every Song ID you search, you can:

* View and set max supply
* View current supply
* View and set split address (royalty splits)

### Rarity Info

For every rarity you search, you can:

* View and set rarity multiplier
* View rarity mint price

### Song Metadata

You can view the song metadata by entering a Song ID and rarity.

### Token Info

For every Token ID you search, you can find the:

* Song Id
* Generation (rarity)
* Metadata
* Owner
* Staking status

### Public Tokens

You can view a list of the tokens that are staked from the contract, and check the token id of each staked token in the list using the index. This lets you see a list of all the public tokens.
