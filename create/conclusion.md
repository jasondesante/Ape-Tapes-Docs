# Conclusion

## Recap of Creation Features

* The Contract Wizard lets artists create contracts they have full control of.  The token has a unique design that makes sense for music.
* The Playlist Creator lets listeners create serverless playlists, unlocking new functionality in the metaverse.
* The Contract Wizard is deployed using 7 new contracts, detailed in [Smart Contracts.](smart-contracts/)
* The Mega Music Metadata Maker is an in depth tool to let you create different pieces of metadata easily, while keeping full control and ownership of your data. &#x20;
* The Manager page is where the owner of a contract goes to do stuff like mint new originals.

## Recap of recap

* 721J contracts make tokens for musicians that have on chain rarity and start as a 1/1. Really important.
* Serverless playlists are really important.
