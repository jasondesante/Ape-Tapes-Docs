# IERC721JEnumerable.sol

The IERC721JEnumerable interface contract details 2 functions. This is inspired by [IERC721Enumerable](other.md#ierc721enumerable.sol), but extended to have functions for an index of staked tokens and an index for each song.

### Read Functions

The new functions are:

* _**tokenOfPublicByIndex**_ - Returns an index of tokens that are set to public
* _**tokenOfSongByIndex**_ - Returns an index of tokens with a particular songId

Read more [here.](https://github.com/jasondesante/721J-Token/blob/master/721J%20Contracts/Interfaces/IERC721JEnumerable.sol)
