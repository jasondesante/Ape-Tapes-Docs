### Write Functions

* **setPromo** - From [IERC721JPromo.](../../interface-contracts/ierc721jpromo.sol.md#write-functions)
* **setPromoSpecific** - From [IERC721JPromo.](../../interface-contracts/ierc721jpromo.sol.md#write-functions)

* **promoMintSpecific** - From [IERC721JPromo.](../../interface-contracts/ierc721jpromo.sol.md#write-functions)

* _**promoMintRarity**_ - Mints a promo just like promoMintSpecific but it only checks for a matching rarity
