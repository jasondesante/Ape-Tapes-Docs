# Minting Functions

## Minting Functions

These functions have to do with minting tokens.  Either minting a new 1/1 original, or minting a copy.

### Mint Original

* **mintOriginal** - From [IERC721J.](../../interface-contracts/ierc721j.sol.md#write-functions)
* _**mintOriginalSplit**_ - Mints an original token with 1 rarity with a royalty split address included.  Just like mintOriginal but it also sets the split address at the same time.
* **mintOriginal3** - From [IERC721J.](../../interface-contracts/ierc721j.sol.md#write-functions)
* _**mintOriginal3Split**_ - Mints an original token with 3 rarities with a royalty split address included.  Just like mintOriginal3 but it also sets the split address at the same time.

### Mint Copy

* **ownerMintCopy** - From [IERC721J.](../../interface-contracts/ierc721j.sol.md#write-functions)
* **mintCopy** - From [IERC721J.](../../interface-contracts/ierc721j.sol.md#write-functions)
* **mintCopyTo** - From [IERC721J.](../../interface-contracts/ierc721j.sol.md#write-functions)
* **mintCopyToken** - From [IERC721JFull.](../../interface-contracts/ierc721jfull.sol.md#write-functions)
* **mintCopyTokenTo** - From [IERC721JFull.](../../interface-contracts/ierc721jfull.sol.md#write-functions)
* **recycleMint** - From [IERC721JFull.](../../interface-contracts/ierc721jfull.sol.md#write-functions)
