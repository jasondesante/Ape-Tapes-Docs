# Price Info

The price tab in The Manager page has all the price info for the contract.

## Balance

You can view the balance of the contract.  You can also withdraw the balance to your wallet as the owner.

## Mint Price

You can view and set the price in ETH or any ERC-20 token.

## Royalties

You can view and set the royalties (based on [ERC-2981](https://eips.ethereum.org/EIPS/eip-2981)), which is a suggested basis points for secondary royalties. &#x20;

Royalty Info lets you calculate the amount of royalties for a secondary sale amount.  This is important because it also includes the address to send it to, so splits work with ERC-2981.  Super cool.
