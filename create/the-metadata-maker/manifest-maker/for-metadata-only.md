# For Metadata Only

The Manifest Maker is intended to be used with metadata files only.  It's meant to make collections of metadata and to be assigned to the baseURI of a smart contract.

### JSON Files Only (or zip)

The only files accepted on this page are metadata JSON files and zips that have JSON files in them.  The zip support is because the song metadata maker outputs a zip when you create many rarities.  So it's easier to add 1 zip file per song than having to unzip the files and add each rarity of each song.  We're trying to make things easier while staying open to customization.&#x20;

The Manifest Maker accepts zip files and will automatically unzip them as long as the zip contains only JSON files.
