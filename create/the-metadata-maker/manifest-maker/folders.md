# Folders

A "manifest" is a shared link for all the files.  Saving lots of gas, and simplifying the urls.

In Arweave, a "manifest" is a special file that defines a folder structure by listing multiple files. You can access any file within the manifest by combining the manifest's transaction ID (tx\_id) with the specific file's path. This allows you to group multiple files under a single URL, with each file accessible via its unique path.

In the context of "The Manifest Maker", that tool is creating a specific type of manifest, that requires only json metadata files meant to use the manifest as the base uri for an nft collection.
