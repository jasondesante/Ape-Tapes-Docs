# Using The Metadata Maker

This page is related to the Song & Profile Metadata Makers.

Making metadata is easy.  First thing is to gather your files.  Fill out the required fields and as many optional fields as you can.

## Adding Files

When adding files to metadata, you can add them from your computer, enter in the url manually, or use The File Picker.

### Drag & Drop

You can Drag & Drop files from your computer to be added to the metadata and uploaded when you save shortly.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-17 at 12.10.53 AM.png" alt=""><figcaption></figcaption></figure>

### File Picker

The File Picker lets you select previously uploaded files to insert into the proper spots in The Metadata Maker.  There's many optimizations using The File Picker, for example you can grab sets of 3 images all at once when you're dealing with an image that needs it's resized variants.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-17 at 12.11.16 AM.png" alt=""><figcaption></figcaption></figure>

### Automatic IPFS

Newly added files will show the IPFS link in their url fields. You'll have the option later to upload to Arweave if you want.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-16 at 11.01.31 PM.png" alt=""><figcaption></figcaption></figure>

## Saving Metadata

You've added files and filled out the form. Now you're ready to save.

### Saving

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-16 at 11.05.12 PM.png" alt=""><figcaption><p>The save button is on the top right of the metadata maker pages</p></figcaption></figure>

Click save when you're ready.  Once you click save, you'll see the options to upload. &#x20;

[Add a license](adding-licenses.md), add tags, and upload to Arweave. Or skip uploading and self host using IPFS.

It's highly recommended to upload your files to Arweave. To do that you'll need to have enough Turbo Credits. You can buy Turbo Credits with card or crypto, or ask a friend to share some credits with the credit sharing feature.

Once you sign, the files are uploaded.  The IPFS links are replaced with the new Arweave links, and the metadata files are created. &#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-16 at 11.09.24 PM.png" alt=""><figcaption></figcaption></figure>

### Downloading

Once the files are uploaded, you'll see "Metadata Created Successfully".

You'll then see two download buttons, one to download every rarity in a zip, "Download All Metadata", and the other to download the raw metadata, which has every rarity in one file. The raw file is there to recover as a save state when updating the metadata later.&#x20;

The "View in Library" button goes to the Arweave Uploads Page where you can see the local metadata files you just created and choose to keep upgrading, upload, or go to The Manifest Maker and create a folder for your collection.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-16 at 11.08.08 PM.png" alt=""><figcaption></figcaption></figure>

## Settings Tab

The Settings Tab is the final tab in both the Song Metadata Maker and Profile Metadata Maker.  The Settings Tab looks the same in both.

You can import old metadata and it will load it into the form.  You can clear the form.  You can read more information with the "Show Help Information" button, and you can read the full Terms & Conditions too.
