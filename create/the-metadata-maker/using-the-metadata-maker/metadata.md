# Metadata

## What is a JSON

JSON stands for JavaScript Object Notation. A JSON Object is a fancy term for a fancy text file with lots of brackets and stuff.

## What is a Metadata File?

Metadata files are the most important piece of data for most NFTs.  This is where the information for everything is.  Everything from the name, description, and attributes, to the images, audio files, videos and more.  You wouldn't be able to keep track of all of that stuff without a properly made metadata file.

Metadata files are JSON Objects.  It's very simple and organized, anyone can read a json file. Every NFT that has off chain data uses a json file for its metadata.

Read more about metadata standards [here](../metadata-standards/).
