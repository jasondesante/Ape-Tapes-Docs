# The Metadata Maker

You can do three main things with the metadata maker:

* Create Song Metadata [here](song-metadata.md)
* Create Profile Metadata (contractURI) [here](profile-metadata-contract-uri.md)
* Update your Collection Manifest [here](manifest-maker.md)

Metadata files are [.json file type](concepts/what-is-a-metadata-file.md).

### Before You Start

Gather your files and prepare all the information you want to include.  Being prepared will make things easy. &#x20;

### Arweave Uploading

You can upload your files and metadata to Arweave using the metadata maker.  Uploading to Arweave with an Ethereum wallet is handled by Irys.



## What Is Arweave?

Arweave is a blockchain used for storing files.  Learn more [here](https://www.arweave.org/) [and](https://ar.io/) [here](https://ao.arweave.dev/).  Check out [Viewblock](https://viewblock.io/arweave) for cool info about the weave.

### Irys

Irys is a layer 2 for Arweave that lets you use an Ethereum wallet to sign Arweave uploads. &#x20;

It works by your wallet having a balance in the Irys node, so you only need to deposit once and can sign for as many uploads as your balance can handle.  You can deposit into the Irys smart contracts on Eth, Base, Polygon, Optimism, and more. &#x20;

Testnet is also supported on Irys. Eth Sepolia, Base Sepolia, and Polygon Amoy networks were tested on The Contract Wizard.  So you can try everything out for free before committing to mainnet.
