# The Wizard's Library

The Contract Wizard's Library is the name for the larger list of all the contracts that have been made by The Contract Wizard.  In technical terms all the contracts in the Wizard's Library have the same "abi".&#x20;



## Together Strong

Any feature made for one contract will work for all contracts in the library.  This idea can be seen in The Store Page.  It treats every contract in the library the same.  The Store Page shows all the tokens that are staked across all contracts in the Wizard's Library.  It's possible for anyone to build new things that work for every contract in the list.



## Decentralized Autonomous Music Network

The Contract Wizard's Library is an on chain, interoperable piece of an open source decentralized autonomous music network. In English that means its a list of record labels.  Crypto record labels. &#x20;

Uniquely owned by each artist, while the whole set can be treated as one large set of collections.  If you do something for one, it works for all.  It's a place that automatically organizes a decentralized group of artists, while letting each artist do exactly what they want. &#x20;
