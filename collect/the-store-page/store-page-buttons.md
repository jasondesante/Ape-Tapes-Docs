# Store Page Buttons

Every token in The Store Page has the same buttons to interact with each token.

## Copy Button

You can find the copy buttons for each song on the right side of the screen.

{% embed url="https://arweave.net/ORWK9VXkyEDX_MmGJDU7p8LJ-EMhi6hrjM4I2I6c1n0" %}

When you click the copy button, you'll see the options to create a copy.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-07-21 at 12.29.45 AM.png" alt=""><figcaption><p>This is what the popup looks like to mint a copy</p></figcaption></figure>

When the window pops up, it'll show an image preview of the token you'll be creating, the mint price and remaining supply of the song.  You can choose to mint the copy to your wallet or some else's.

The reason you want to mint directly to someone's wallet that isn't yours is to honor the spirit of copying tapes and cds to friends to share the love of music.  If you like a song you'd want to share it by minting copies to your friends.  Or maybe you want to copy trade with someone who has a token you don't have and trade copies to each other.

### Recycling

One special option for minting is Recycling.  This feature needs to be enabled, and allows you to mint a copy by recycling 2 tokens that already exist.  With recycling you are reducing the supply by burning 2 tokens that are less desirable in exchange for 1 new token.  This only makes sense in certain setups but is useful for balance.  That is why it is an opt in feature that needs to be enabled.

## Info Button

To the right of the copy button is the info button.  It's icon looks like the Contract Wizard's Hat.  This is because the magic hat represents knowledge and information.

{% embed url="https://pduxa3myz6uqwffz45fces4a3ociwp364s6iiuvxnxt3d3l7pjla.arweave.net/eOlwbZjPqQsUuedKIkuA24SLP37kvIRSt23nse1_elY" %}

When you click the info button a window will pop up.

<figure><img src="../../.gitbook/assets/Screenshot 2024-07-09 at 4.12.14 PM (1).png" alt=""><figcaption><p>This is what the info popup looks like</p></figcaption></figure>

This shows you more info about the token.  This includes the name, artist, Token Id, Song Id, and generation (rarity).  The current supply, contract address and metadata url are also listed.

The wallet that owns the token you're minting a copy of is listed.  This address can be an unknown wallet, or a well known wallet.  This might be important if you care about the provenance of the family tree.
