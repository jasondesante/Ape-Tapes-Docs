# Settings

## Settings

The settings page has some options you might need to use like:

#### Pages

* Records Page
* Achievements
* Welcome Page
* Show Credits

#### User Settings

* Log In / Out
* Toggle Hat Helper: Off/On
* Save Audio: Off/On
* Audio Quality: Quality/Fast
* Graphics Quality: Pretty/Fast
* Track Images: Show All/Hide Unknown

#### Tools

* Arweave Explorer
* Configure Embed Player
* Webamp Toy - Classic Winamp player with skins that supports Serverless Playlists.

#### Network

* Switch Chains from Mainnet to Testnet&#x20;

#### Extras

* Delete some or all your data
* Set custom colors
* Secret bonus settings
