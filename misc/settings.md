# Settings

## Settings

The settings page has some options you might need to use like:

* Log In / Out
* Toggle Hat Helper
* Configure Embed Player
* Switch Chains from mainnet to testnet&#x20;
* Delete some or all your data
* View Welcome page again
* Secret bonus settings



The testnet version of the site is important because when you are on testnet you can try everything for free.&#x20;
